Qu'est-ce que l'outil Octopus ?

Octopus est un outil générique de cassage de mots de passe distribué.

Comment utiliser Octopus ?

Via cette interface Web, ou via l'API

Comment ajouter des types d'attaques ou des programmes ?

Il faut avoir en tête six éléments.

attaque -> [is_att] -> True/False (à ajouter dans attacks.py dans jobParser{})

attaque -> [att_chunkify] -> liste de chunks (à ajouter dans chunkify.py dans typetoChunker)

chunk -> [get_chunk] -> extended_chunk (à ajouter dans chunkify.py dans getChunk{}, sert notamment à ne pas avoir à manipuler de grosses données par exemple dans le cas de dictionnaires volumineux à utiliser)

hash -> [format] -> formatted_hash (à ajouter dans attacks.py dans makeHash{})

vars -> [attack] -> command (à ajouter dans attacks.py dans commands{})

regexes à ajouter dans params.py (replace(regex1, regex2) doit donner "mdp:[hash|username]"
